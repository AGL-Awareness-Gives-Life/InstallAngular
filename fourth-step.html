    1. What is Visual Studio Code?

    Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js

    2. How to Download and install Visual Studio Code?

    You can Visit here, VSCode and download the stable version of Visual Studio Code

    3. How to open VSCode from Command Prompt?

    Open visual Studio Code and go to View Menu click on Command Palette

    Serach for Install Code command in path and install it.

    With this you can go to command prompt and use code command to open any folder with VSCode.

    If you don't like VSCode editor, feel free to use any Angular Enabled editors like Brackets , Sublime Text Atom , etc.
