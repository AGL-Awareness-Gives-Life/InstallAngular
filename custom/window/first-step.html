  • Create the project structure

    1. The easiest way to create Angular project need two folders.

    Main folder will contain all project related files, as well as an "app" folder inside it.

    Name Discription
    example This is the root folder and it have all the project related files and configration files.
    example/app This folder contains Angular code files.

    2. Create a HTML document in root folder.

    After creating HTML file, add the content given below:

    <!DOCTYPE html> <html> <head> <title>Angular</title> <meta charset="utf-8" /> </head> <body> <app>This is static content</app> </body> </html>

    "app" is a custom element to show dynamic content.
