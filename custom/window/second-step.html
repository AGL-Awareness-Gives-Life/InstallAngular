  • Create "package.json" and Add dependencies

    1. Create package.json

    package.json is used by NPM for its configration file, there are three section

    Name Discription
    dependencies This is the section to specify the packages that angular application depends on, including angular.
    devDependencies This is the section to specify the packages that we used in development, such compilers and test frameworks.
    scripts This is the section to specify the commands that we use to compile, test or run the application.

    { "dependencies": { }, "devDependencies": { }, "scripts": { } }
