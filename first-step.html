  • Time to Download and Install NodeJS!

    1. What is NodeJS?

    Node.js® is basically a runtime environment for executing JavaScript code outside the browser. Node provides us useful tools like NPM, which are needed for developing Angular Projects.

    2. How to Download NodeJS?

    Visit the site Nodejs.org and choose stable version (Recommended For Most Users) to download NodeJS Installer.

    3. How to Verify the nodeJS version?

    Open Up command prompt and type "node -- version". You should be able to see the nodeJS version.

    Minimum version required for angular is 6.9.x
