  • Use NPM tool!

    1. What is NPM?

    Node.js provides a tool called Node Package Manager(NPM). It is used to install dependencies required for angular project.

    2. Why Node Package Manager(NPM)?

    It is the standard mechanism for managing JavaScript packages.The world of JavaScript development has fragmented into small packages that together help to create rich and fluid functionality.Manually downloading packages required for project means working through a long process of downloading,installing and inspecting each package.

    3. Verify the NPM version?

    Open command prompt and type "npm -- version" to check for version of NPM installed on system.
